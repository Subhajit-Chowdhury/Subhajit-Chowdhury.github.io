# Personal Portfolio: Subhajit Chowdhury

**Live Deployment:** [subhajit-chowdhury.github.io](https://subhajit-chowdhury.github.io)

This repository holds the source code for my professional data engineering portfolio. Built natively with HTML, CSS, and Bootstrap, it operates as a high-speed, zero-dependency static site designed to instantly deliver my professional experience, project documentation, and technical stack to recruiters and engineering managers.

## 🚀 Architecture & Development

The primary goal of this repository is to guarantee highly reliable, sub-second content delivery across all devices with zero framework overhead.

* **Engineered the core layout** using native HTML5 and CSS3, leveraging the Bootstrap 4 grid system to construct a fully fluid and responsive viewport experience seamlessly scaling from 4K desktop environments down to mobile devices.
* **Implemented custom CSS animations** (including unified scaling transforms and dynamic vector color mapping natively triggered on hover) to increase UX engagement without sacrificing client-side rendering speed.
* **Integrated global CDN assets** via FontAwesome and DevIcons to securely serve high-fidelity, scalable SVG technology logos (AWS, Snowflake, Python, SQL) while minimizing local repository bloat.
* **Automated production hosting** by bridging the `main` branch directly to GitHub Pages, ensuring instantaneous, zero-downtime deployment for any content or UI updates.

## 📊 Technical Content & Scope

This portfolio serves as the unified hub for my enterprise data engineering footprint, summarizing nearly 4 years of scalable architecture delivery.

* **Quantified Enterprise Impact:** Documents the architecture and maintenance of 150+ production-grade ETL/ELT pipelines operating at MB–TB scale daily across the exact AWS ecosystem (S3, Glue, EMR, Redshift, Athena).
* **Highlighted Core Data Principles:** Showcases distinct implementation of Medallion Architectures (Bronze/Silver/Gold layers) and modern data lakehouse designs integrating Delta Lake and Snowflake.
* **Detailed End-to-End Serverless Projects:** Maps out personal project codebases containing continuous, low-latency streaming (EventBridge, DynamoDB Streams) and batch analytics processing utilizing Snowpipe and AWS Glue Crawlers.

## ⚙️ Local Development

To run this portfolio locally or audit the CSS structure:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Subhajit-Chowdhury/Subhajit-Chowdhury.github.io.git
   ```
2. Navigate into the directory and launch any standard local web server (e.g., Python's `http.server` or VS Code Live Server) to bypass cross-origin browser restrictions:
   ```bash
   python -m http.server 8080
   ```
3. Access the site locally at `http://localhost:8080`.
